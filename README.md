Due to the influx of AI generated plugins, please review the following guidelines before you submitting Pull Requests for pluginList.json:

1) Announce your plugin to the community via https://www.facebook.com/groups/1554782254796752 and receive & implement feedback
2) Test & confirm your plugin functions correctly on both the last released FPP version and the current nightly image
3) Ensure your plugin conforms to the FPP UI look/feel.
   a) Creates no more than 1 item in each of the Status/Control, Content Setup, Input/Output Setup menus
   b) Ensure your plugin reflects FPP's general UI look feel.  In most cases, this means retaining the top FPP menus and bottom status bar.
4) Plugins that require subscriptions to unlock plugin or related website features are strongly discouraged.
5) All plugins are accepted purely at the discretion of the FPP developers and are subject to removal without notice at any time.
